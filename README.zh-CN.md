# ExecManager

[English README](README.md)

ExecManager 是一个面向 Kimi Code 的本地集成层。

它提供一个 installer-grade 入口，用来先把二进制安装到本机，再由 `execmanager` 自己完成本地环境配置。

## 安装

```bash
curl -fsSL https://raw.githubusercontent.com/xycld/execmanager/main/install.sh | bash
execmanager
```

## 使用

命令和参数请直接查看：

```bash
execmanager -h
```
